/nodes/{nodeid}/bridges post get

get

post /nodes/{nodeid}/bridges

Creates a new bridge

  • Request
  • Response

URI Parameters

  • nodeid: required (string)

Annotations

  • AYSBased: ()

Body

Media type: application/json

Type: object

Properties
  • name: required (string)

    Name for the bridge to be created

  • hwaddr: (string)

    MAC address for the bridge to be created. If none, a random MAC address will be assigned.

  • networkMode: required (one of none, static, dnsmasq)

    Networking mode

  • nat: required (boolean)

    If true, SNAT will be enabled on this bridge.

  • setting: required (object)

    Networking settings are depending on the selected 'networkMode' as follows:

    • networkMode = 'none': No settings Bridge won't get any IP settings.
    • networkMode = 'static': settings={'cidr': 'ip/net'} Bridge will get assigned the given IP address.
    • networkMode= 'dnsmasq': settings={'cidr': 'ip/net', 'start': 'ip', 'end': 'ip'} Bridge will get assigned the IP address specified for 'cidr' and each running container that is attached to this IP address will get an IP address within the specified start/end range. Netmask of the range is the netmask part of the provided CIDR. If 'nat' is set to true, SNAT rules will be automatically added in the firewall.
    • cidr: required (string)
    • start: required (string)
    • end: required (string)

HTTP status code 201

Bridge created successfully

Headers

  • Location: required (string)

    URL to the created bridge resource
