/node/{nodeid}/bridge post get

post

Creates a new bridge

get

List bridges

post /node/{nodeid}/bridge

Creates a new bridge

  • Request
  • Response

URI Parameters

  • nodeid: required (string)

Body

Media type: application/json

Type: object

Properties
  • name: required (string)

    Name for the bridge to be created

  • hwaddr: (string)

    Macaddress for the bridge to be created. If none, a random macaddress will be assigned

  • networkMode: required (one of none, static, dnsmasq)

    Networking mode

  • nat: required (boolean)

    If true, SNAT will be enabled on this bridge.

  • settings: required (object)

    Networking settings, depending on the selected mode. none: no settings, bridge won't get any ip settings static: settings={'cidr': 'ip/net'} bridge will get assigned the given IP address dnsmasq: settings={'cidr': 'ip/net', 'start': 'ip', 'end': 'ip'} bridge will get assigned the ip in cidr and each running container that is attached to this IP will get IP from the start/end range. Netmask of the range is the netmask part of the provided cidr. if nat is true, SNAT rules will be automatically added in the firewall.

    • cidr: required (string)
    • start: required (string)
    • end: required (string)

HTTP status code 201

Bridge created successfully

Headers

  • Location: required (string)

    Url to the created bridge resource
